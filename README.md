# AES-files-Encryptor-Decryptor
A script that encrypts & decrypts files using AES-128-ECB, AES-192-ECB or AES-256-ECB algorithms.

## Features
This script provides the following features:

- 128 bit keys
- 192 bit keys
- 256 bit keys
- ECB Mode
